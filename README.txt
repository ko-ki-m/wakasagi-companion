Wakasagi Map v5

目的:
- GitHub Pages / ホーム画面固定起動でスマホGPSを取得
- 起動直後に現在地を地図でピンポイント拡大表示
- スマホ内DBの過去釣行ポイントを地図上に表示
- 20m以内 / 100m以内の過去釣行回数を表示
- 過去ポイントの詳細: 日付、湖名、ポイント名、ライン、シンカー、魚探水深、水温、メモ

削除したもの:
- 独自の釣行開始/終了
- 独自sid運用
- ヒット記録ボタン
- event_markersの新規利用

残したもの:
- 現在地取得
- ポイント保存
- 過去ポイント照合
- Export/Import

GitHub Pages上の既存ファイルを、このフォルダの中身で上書きしてください。
サービスワーカーのキャッシュ名は wakasa-map-v5 です。
