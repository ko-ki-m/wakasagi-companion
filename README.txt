Wakasagi Map v7 FIELD

v7の目的:
- v5/v6で表示できた地図とポイントを、実釣で使える管理画面へ進めた版。
- Pico W本体スケッチには触れない。
- 独自sid、独自釣行開始、ヒット記録ボタンは追加しない。

v7の追加点:
- 現在地中心に20m同一ポイント円、100m同一エリア円を表示。
- 周辺全体ボタンで現在地と100m圏内ポイントをまとめて表示。
- 20m以内に既存ポイントがある状態で新規保存する時は重複確認。
- 選択した過去ポイントを編集フォームへ読み込み、座標を変えずに湖名/ポイント名/ライン/シンカー/魚探水深/水温/メモを上書き可能。
- Exportのversionを7へ更新。
- Service Workerキャッシュ名を wakasa-map-v7-20260507 に更新。

GitHub Pages更新手順:
1. このフォルダ内の全ファイルを、GitHub Pagesで公開している場所の既存ファイルに上書きする。
2. Safariで https://.../reset.html を1回開く。
3. 自動で index.html?v=7 に移動し、画面上部が Wakasagi Map v7 になることを確認する。
4. その後、必要ならホーム画面アイコンを作り直す。

重要:
- DB名は wakasa_companion_v2 のままなので、v5/v6で保存したポイントは維持します。
- event_markersや独自ヒット記録は復活させていません。
