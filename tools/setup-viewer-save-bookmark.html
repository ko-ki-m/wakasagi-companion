ワカサギ viewer保存ブックマーク作成

PC Chrome / Edge では、下の黒いリンクをブックマークバーへドラッグします。

ワカサギ viewer保存

このリンクをクリックするのではなく、ブックマークバーへドラッグして登録します。

ブックマークバーが出ていない場合は、PCで Ctrl + Shift + B を押して表示します。

登録後、Pico W の /log 画面を開いた状態で、ブックマークバーの「ワカサギ viewer保存」を押します。

ドラッグできない環境用：下のボタンでURL欄用コードをコピーします。

正しい登録後、ブックマークバーに表示される文字は「ワカサギ viewer保存」だけです。

長い javascript:(async... が名前として見えている場合は登録失敗です。
